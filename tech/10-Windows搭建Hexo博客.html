Windows搭建Hexo博客

1. 安装Git

下载 git 并完成安装 , 后面下载项目主题或者发布博客到GithubPages会用到

https://git-scm.com/download/win


2. 安装Node.js

下载 Node.js 并完成安装

https://nodejs.org/en/


3. 安装Hexo

运行CMD , 输入 npm 命令安装Hexo

npm install -g hexo


4. 创建Hexo项目

继续在命令行中 cd 到准备创建Hexo项目的文件夹 , 比如 E:\Hexo , 运行创建命令 :

hexo init

创建完可以看到该文件夹已经生成了Hexo相关文件 , 继续安装依赖包 , 运行命令 :

npm install

然后生成网站 , 运行命令 :

hexo generate

然后启动Hexo服务 , 运行命令 :

hexo server

成功后命令行提示如下 :

INFO  Start processing
INFO  Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.

表示Hexo服务已成功启动 , 在浏览器输入 http://localhost:4000/ , 即可访问生成的博客页面

Hexo生成的是静态的博客 , 也就是静态的Html文件 , 因此我们可以直接在本地用浏览器浏览 , 生成的目录是在 \Hexo\public , 将该文件夹里面的文件复制到磁盘根目录 , 打开 index.html 文件 , 即可浏览 , 这也是可以直接用 GithubPages 搭载 Hexo 的原因所在


5. Tips

Hexo 生成网站运行默认用的是 4000 端口 , 如需要指定端口 , 运行如下命令 :

hexo server -p 3000

本地Hexo文件错生成乱时 , 运行清理命令 :

hexo clean

Hexo可以使用更简洁的命令 , 如下 :

hexo g   =>   hexo generate
hexo d   =>   hexo deploy
hexo s   =>   hexo server
hexo n   =>   hexo new
