Eclipse 使用指南

1. 配置 JDK

点击 : Window -> Prefernces -> Java -> Installed JREs -> Add -> Standard VM -> 选择 JDK 目录 C:\Program Files\Java\jdk1.8.0_211 -> Finish -> 在列表勾选刚刚添加的 JDK -> Apply

2. 自定义工具栏

点击 : Window -> Perspective -> Customize Perspective -> Tool Bar Visibility , 选择对应的选项即可

3. 安装 SVN 插件

从 Eclipse Marketplace 里面下载安装 , 具体操作 :
打开 Eclipse -> Help -> Eclipse Marketplace -> 在 Find 中输入 subclipse 搜索 -> 找到 subclipse 点击 install 如果打开 Eclipse 弹出 Subversion Native Library Not Available , 点击 : Window -> Prefernces -> Team -> SVN , 在右侧的 SVN 接口 : 的 Client: 选项选择 SVNKit... 选项即可

4. SVN 指定文件不同步

安装好 SVN 插件之后 , 点击 : Window -> Show Window -> Other... -> Team -> Synchronize , 切换到当前项目 , 然后在 Synchronize 窗口右键选择不同步的文件 , 在弹出的菜单中选择 添加至 svn:ignore 即可 , 如果 Synchronize 窗口里面没有当前项目 , 只用在想目处右键 , 选择 Team -> 与资源库同步 即可

5. 去掉 XML 校验

点击 : Window -> Prefernces -> Validation , 去掉 XML 打头的校验选项即可

6. 编辑器字体修改

点击 : Window -> Prefernces -> General -> Appearance -> Colors and Fonts -> Java -> Java Editor Text Font , 个人推荐 Consolas

7. 常量和静态方法设置非斜体

点击 Window -> Preferences -> Java -> Editor -> Syntax Coloring , 在右侧的 Element: 中 选择 static 开头的选项 , 去掉 Italic 属性即可

8. 修改默认编码

  • 点击 : Windows -> Preferences -> General -> Workspace , 在右侧的 Text file encoding 中选择 UTF-8
  • 点击 : Windows -> Preferences -> General -> Content Types , 在右侧点开 Text , 选择 Java Source File , Java Properties File , JSP , 在下面的 Default encoding 输入 UTF-8 , 点击 Update
  • 点击 : Windows -> Preferences -> Web -> JSP Files , 面板选择 ISO 10646/Unicode(UTF-8)
  • 右键选择项目 -> Properties -> Resource -> 设置编码为 UTF-8

9. 设置代码自动提示

点击 Window -> Preferences -> Java -> Editor -> Content Assist , 在右侧的 Auto Activation 选项的 Auto activation triggers for Java 中输入 .abcdefghijklmnopqrstuvwxyz 即可

10. 删除工作空间

点击 Window -> Preferences -> General -> Startup and Shutdown -> WorkSpace , 然后选中你想要删除的工作空间 , 点击右边的 Remove 按钮即可

11. 配置 Maven

首先配置系统环境变量 :
右键计算机 -> 属性 -> 高级系统设置 -> 高级 -> 环境变量 , 在下方的 系统变量 添加 MAVEN_HOME , 值是 Maven 文件夹路径 , 如 : C:\MyProgram\Maven , 修改 Path 变量 , 添加 %MAVEN_HOME%\bin; , 注意分号 ; 配置完成之后查看是否成功 : 运行 CMD 输入 mvn -v
然后配置 Eclipse :
打开 Eclipse -> Window -> Preferences -> Maven -> User Settings -> Global Settings: 和 User Settings: -> Browse... , 选择 Maven 目录下的 \conf\settings.xml 文件 , 然后点击 Update Settings 按钮即可
Maven 修改库目录 :
打开 \conf\settings.xml 文件 , 找到 <localRepository> 标签 , 值修改为目标路径即可

12. 配置默认 JDK 版本

点击 Window -> Preferences -> Java -> Compiler , 一般勾选默认的就可以

13. 去掉 JS 校验

点击 Window -> Preferences -> Java Script -> Validator -> Errors/Warnings , Enable Javascript Sematic validation 前面的勾选去掉 ;
右键项目 -> Properties -> Builders , 去掉 JavaScript Validator 前面的勾

14. 关闭拼写检查

点击 Window -> Preferences -> General -> Editors -> Text Editors -> Spelling , Enable spell checking 前面的勾选去掉即可

15. Maven 下载源码

打开左上角的 Terminal , 默认执行路径是当前项目 , 运行如下命令 : 

# mvn dependency:sources
