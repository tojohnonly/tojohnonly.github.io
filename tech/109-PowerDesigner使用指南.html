PowerDesigner 使用指南

1. Name Code 取消同步

Tools -> General Options... -> Dialog -> Name to Code mirroring -> 取消勾选 , 确定即可

2. 设置字体

字体设置是和项目走的 , 也就是不同的项目 , 不同的表字体都可能不一样 , 具体设置如下 :

  1. 选择项目 -> Tools -> Display Preferences... -> Table -> Format -> Modify... -> Font
  2. 在左边的 Symbol 列表里选择需要修改字体的项目 (我都是全选) , 然后在右侧的 Font 里修改字体大小样式
  3. 点击 应用 , 确定 , 返回到前一个界面时 , 点击 Set As Default , 点击 OK
  4. 然后在弹出的对话框中选择应用到全部的标志上即可

3. 删除分割线

Tools -> General Options... -> General Settings -> General -> Diagram -> Show page delimiter -> 取消勾选 , 确定即可

4. 自定义显示列

用 Code 列做例子 :
Tools -> General Options... -> General Settings -> Table -> Content -> Advanced... -> Columns -> List columns 右侧的过滤图标
在 Code 项点击勾选 , 然后上移到指定位置 , 最后确定用到所有标志即可

5. 同时显示中文表名和英文表名

PowerDesigner 默认是不支持这么干的 , 但是我们可以把 Name 改成 中文加英文的 , 如 t_bd_service (服务信息表)

6. 创建 UML 类图

File -> New Model -> Categories -> Information -> UML Class Diagram
